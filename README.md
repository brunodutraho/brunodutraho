# 🚀 Projeto em Destaque

## 📊 Marketing Analytics Data Pipeline

Pipeline completo de análise de dados de marketing desenvolvido com **Python, Pandas, PostgreSQL, Docker e Metabase**, simulando um fluxo de trabalho moderno de análise de marketing.

O projeto transforma **dados brutos de campanhas de marketing em tabelas analíticas e dashboards interativos**, permitindo análise de desempenho de campanhas e geração de insights de negócio.

🔗 Repositório  
https://github.com/brunodutraho/marketing-analytics-pipeline

---

## 🏗 Arquitetura do Pipeline

Fluxo simplificado do pipeline analítico:
````
Raw Marketing Data (CSV)
↓
Data Processing (Python + Pandas)
↓
Data Warehouse (PostgreSQL - Star Schema)
↓
SQL Analytical Views
↓
Business Intelligence Dashboard (Metabase)
````

Esse fluxo simula uma arquitetura simplificada de análise utilizada em ambientes corporativos orientados a dados.
